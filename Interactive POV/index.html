Interactive POV Demo

This is a demo of interactive story-telling, the aim of this project was to produce a full experience of up to 12-15 minutes long. Limitations were discovered towards the end of the implementation process, this is spoken about in the artefact report. Although the aim of the project was to be 12-15 minutes long, the most important part of the project was exploring technology used in interactive story-telling.
